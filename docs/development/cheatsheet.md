# Cheatsheet

## Building

```bash
# Requires Rust stable toolchain
cargo build
cargo build --release

# Run all tests
cargo test --all

# Run specific suites
cargo test --test conformance
cargo test --test integration
cargo test --test integration_runtime_network_storage
cargo test --test integration_resource_orchestration
cargo test --test smoke

# Run the binary (requires --node-name)
./target/debug/kubelet --node-name my-node --help
```

## Running Benchmarks

```bash
# All benchmarks
cargo bench

# Specific suites
cargo bench --bench pod_operations
cargo bench --bench memory_profile
cargo bench --bench server_throughput

# Save baseline (for future comparisons)
cargo bench -- --save-baseline main

# Compare against baseline
cargo bench -- --baseline main

# Filter to specific benchmark
cargo bench --bench pod_operations "pod_phase"

# HTML reports (in target/criterion/)
# Automatically generated by criterion
```

## Local Colima E2E

Use [hack/e2e/colima-run.sh](hack/e2e/colima-run.sh) to provision a single-node Kubernetes cluster in a Colima VM (containerd + kubeadm + Calico) and run the full test suite.

```bash
bash hack/e2e/colima-run.sh
```

Skip the binary rebuild and repo sync if you've already built:

```bash
SKIP_BUILD_BINARY=1 bash hack/e2e/colima-run.sh
```
